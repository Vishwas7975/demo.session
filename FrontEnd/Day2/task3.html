  2. unordered list
  3. data defintion/ description list
  4. nested list

Ordered list

It is used to display number list/in a specific order

ol tag is used to create a ordered list

The default order is number (1,2,3...)

Type attribute helps to change the default order(Type="A","a","1","I","i")

Reversed attribute helps to reverse the order

Start attribute helps to start the list form a specific number

Data description list/ description list

It is used to display a list of terms and their description

dl tag is used to create a data definition

dt tag is used to define a term

dd tag is used to define a description
