Selector:

It is used to select HTML elements for styling purpose
  • element selector
  • id selector
  • class selector
  • attribute selector
  • grouping selector
  • Universal selector
  • combined selector
    • descendent selector
    • child selector
    • adjacent selector
    • general sibling Selector

Element Selector:

It is used to select elements by their tag name
        syntax:       tagname {
                          property:value;
                        }

Id Selector:

It is used to select elements by their id

Id is an attribute used in opening tag

Id should be unique

Id selector indicated with #

            syntax:      #idName {
                              property:value;
                            }

Class Selector:

It is used to select elements by their class

class is an attribute used in opening tag

class name should be duplicate

class selector is indicated with .

        syntax:          .class1 {
                              property:vlaue;
                            }

Grouping Selector:

It is used to select elements as grouping

grouping selector indicated with ,

It is used to apply same style to multiple elements

        syntax:         selector1,selector2, selector3 {
                                                   property:value;
                                                }

Combined Selector:

It is used to combine multiple selectors

It is used to apply same style to multiple elements

Descendent Selector:

It is used to select elements that are descendent of a specified element

Used to select both direct and indirect child element

descendent selector indicated with a space

        syntax:        parent child {
                          property:value;
                       }

Child Selector:

It is used to select elements that are direct children of a specified element

Used to select only direct child element

child selector indicated with a (>) 

        syntax:        parent>child {
                          property:value;
                       }

Adjacent Sibling Selector:

It is used to select elements that are adjacent sibling of a specified element

Used to select only the next sibling element

adjacent sibling selector indicated with a (+)

        syntax:        element1 + element2 {
                          property:value;
                       }

General Sibling Selector:

It is used to select elements that are general siblings of a specified element

Used to select all sibling elements

general sibling selector indicated with a (~)

        syntax:        element1 ~ element2 {
                          property:value;
                       }

Universal Selector

It is used to select all elements

Universal selector indicated with *

        syntax:       *{
                        property: value;
                    }
        Note : it is recommended to have 
        padding :0;
        margin:0;
        box-sizing: border-box;
        content-box-it is used to set the box model by default
                   -it calculates the width and height of an element
                    including padding and border
                    ex: width: 100px;
                        height: 100px;
                        padding: 10px;
                        border: 2px solid black;
                        box-sizing: content-box;
                    calculation:
                        width: 100px + 10px + 10px + 2px + 2px = 124px;
                        height:100px + 10px + 10px + 2px + 2px = 124px;
                        
     box-sizing: border-box-it is used to set the box model by manually
                -it calculates the width and height of an element 
                including padding and border and provides the exact width and height
                ex: width: 100px;
                    height: 100px;
                    padding: 10px;
                    border: 2px solid black;
                    box-sizing: border-box;
                calculation:
                    width: 100px + 10px + 2px = 100px;
                    height: 100px + 10px + 2px = 100px;
